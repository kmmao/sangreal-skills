# 示例技能

这是一个简单的示例技能，用于演示 Claude Skills 的基本结构。

## 用途

此技能演示了：
- 如何编写技能定义文件
- YAML frontmatter 的格式要求
- 如何组织技能指令

## 使用说明

当用户提到"使用示例技能"或"演示技能功能"时，Claude 会激活此技能并遵循以下指令：

1. 向用户解释技能的基本结构
2. 展示如何创建自定义技能
3. 提供技能开发的最佳实践

## 示例

用户可以这样使用：
- "用示例技能给我讲解一下"
- "示例技能是怎么工作的？"
- "帮我创建一个类似的技能"

## 指南

创建自定义技能时请遵循：
- 在 frontmatter 中明确定义 `name` 和 `description`
- 描述要清晰，让 Claude 知道何时激活此技能
- 提供详细的指令，包括使用场景和示例
- 保持指令简洁明了，避免过于复杂
